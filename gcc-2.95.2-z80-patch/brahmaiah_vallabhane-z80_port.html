Z80 port status:
-----------------
It can handle

1) Move operations (loads/stores).
2)Arithmetic operations. (Add, Subtract,shift and not
mul/div (libcalls))
3)Logical operations (AND,IOR, XOR and Shifts)
4) Compares and braches

  I am glad to say that the basic port is up.

In z80.h The MAX_FIXEDMODE_SIZE is 32. (long, long
long and long double are also
32 bits).

The work needs to be done on:
-------------------------------
1) Getting optimized code. When compiled with -Os cc1
aborts because of some
reasons. Need to fix them
2) COST macros need to be implemented with suitable
values. As of now I am
having some default values.
3) Need to add some peep hole optimizations (if
applicable).
4) The compiler's functionality is yet to be  tested.
I have some test programs,
I verified the code with them.
(Rather I am not botherd much about functionality now,
It should not break
when compiling some code.)
